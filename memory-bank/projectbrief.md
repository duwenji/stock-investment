# 株式投資アシスタント - プロジェクト概要

## プロジェクトの目的
生成AIを活用した株式投資アシスタントシステムの開発。投資判断のサポート、ポートフォリオ管理、リスク評価をAIで支援します。

## 主要機能要件
1. **株式データ管理**
   - 株価データの収集と保存
   - 銘柄情報の管理
   - 時系列データの分析

2. **投資判断支援**
   - AIによる買い/売り判断の提案
   - テクニカル分析の自動化
   - リスク評価の提供

3. **ポートフォリオ管理**
   - 保有銘柄の追跡
   - 損益計算
   - 資産配分の最適化提案

4. **取引計画管理**
   - 取引条件の設定
   - 監視ポイントの管理
   - 自動アラート機能

## 技術要件
- **データベース**: PostgreSQL (MCPサーバ経由)
- **AI統合**: 生成AIの活用
- **データソース**: 株価API連携
- **分析機能**: テクニカル指標計算

## 主要ユーザー
- 個人投資家
- 投資初心者
- ポートフォリオ管理者

## 成功基準
- 投資判断の精度向上
- ポートフォリオ管理の効率化
- リスク管理の自動化
- ユーザーフレンドリーなインターフェース
